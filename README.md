# 🔐 Secure Password Generator

## 📌 Project Overview
A simple yet effective Python program that generates strong, random passwords based on user-defined length. Developed as part of a university admission portfolio to demonstrate proficiency in Python programming, logic, and user input handling.

## 🛠 Features
- Generates passwords using letters, digits, and special characters.
- Input validation to ensure minimum security length.
- Clean and user-friendly command-line interface.

## 💻 Technologies Used
- Python 3.x
- Standard libraries: `random`, `string`

## 🚀 How to Run
1. Ensure Python 3 is installed on your system.
2. Clone or download this repository.
3. Open your terminal or command prompt in the project folder.
4. Run the script using the following command:
   ```bash
   python main.py
